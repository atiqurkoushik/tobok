## 📖 About

**Tobok Foods** is a premium Arabic sweets and bakery restaurant chain with three convenient locations in Dhaka, Bangladesh. We specialize in authentic Saudi Arabian sweets, fresh bakery items, creamy doi (traditional yogurt), and savory snacks — all crafted with passion and precision.

🍽 **Perfect for:** Occasions, get-togethers, milad mahfils, gifting, and family gatherings  
💰 **Experience:** Middle-Eastern sweetness right here in Dhaka  
🌟 **Promise:** Fresh, premium, and unforgettable in taste

## 🚀 Getting Started

### Prerequisites

- PHP 8.0 or higher
- Web server (Apache, Nginx, or built-in PHP server)
- Modern web browser

### Local Installation

1. **Clone or download the repository**
   ```bash
   git clone <repository-url>
   cd tobok
   ```

2. **Start PHP built-in server** (for quick testing)
   ```bash
   php -S localhost:8000
   ```

3. **Open your browser**
   ```
   http://localhost:8000
   ```

4. **For production deployment:**
   - Upload all files to your web hosting
   - Ensure `img/` directory contains:
     - `logo.png` (company logo)
     - `slide1.jpg`, `slide2.jpg`, `slide3.jpg` (hero images)
   - Configure virtual host if needed

## 🏗 Technical Architecture

### Backend
- **Language:** PHP 8+ (server-side rendering)
- **Session Management:** PHP sessions for cart functionality (expandable)
- **Routing:** Simple file-based routing (index.php)

### Frontend
- **HTML5:** Semantic markup
- **CSS3:** Custom properties (CSS variables), Flexbox, CSS Grid, Animations
- **JavaScript (Vanilla):** Slider functionality, mobile menu toggle
- **External Resources:**
  - Font Awesome 6.4.0 (icons)
  - Responsive across all devices

### Performance Optimizations
- Preload critical images (logo, hero slides)
- CSS-only animations (GPU-accelerated)
- Mobile-first responsive design
- Minimal external dependencies
